# Spanish Tracking Website

A website that visualizes my Spanish study progress using data pulled from Google Sheets.

## Project Goal

Display charts and summaries showing how much Spanish I've studied over time — covering listening, reading, and speaking activity.

## Data Source

All data comes from this Google Sheet (updated daily):
https://docs.google.com/spreadsheets/d/1SH2D4hFjkevn6iq8wBwh79_Ei2jQX--BxMwcgOTPdzM/edit?usp=sharing

### Tabs

| Tab | Sheet ID | Contents |
|-----|----------|----------|
| Reading | `gid=141435762` | Books/articles I have read |
| Speaking | `gid=21198935` | Speaking session logs — use **duration** as the primary time metric |
| Daily Log | `gid=1655094901` | Daily totals for listening, reading, and speaking (2025 + 2026 YTD) |

## Key Data Rules

- For speaking time, always use the **duration** column as the measure of time spent.
- The daily log tab is the primary source for aggregate time-per-day across all activity types.
