## Vocab / Anki (`vocab/`)

Automates the pipeline from Spanish class notes → example sentences → Anki flashcards.

### Learner Context

- **Level:** B2
- **Focus:** Argentinian Spanish (River Plate dialect, lunfardo, local slang)
- When generating example sentences, prefer Argentine vocabulary and phrasing (e.g. *vos* instead of *tú*, local slang terms).

### Files

| File | Purpose |
|------|---------|
| `vocab/process_class_notes.py` | Main script — reads class notes, generates sentences, updates Total Vocab.md, creates Anki deck |
| `vocab/create_deck.py` | Original script — builds a full Anki deck from all of Total Vocab.md |
| `vocab/processed_notes.txt` | Tracks which class notes have already been processed |
| `vocab/spanish_new_cards.apkg` | Anki deck containing only the most recently added cards |
| `vocab/spanish_vocab.apkg` | Full Anki deck of all vocab (generated by create_deck.py) |

### Obsidian Vault

Located at: `~/Library/Mobile Documents/iCloud~md~obsidian/Documents/Spanish/`

| File/Folder | Contents |
|-------------|---------|
| `Class Notes/` | One `.md` file per class session (named by date, e.g. `03.09.2026.md`) |
| `Total Vocab.md` | Master vocab table: Vocab \| Translation \| Example Sentence |

#### Class Note Format

Each class note has a table with two columns:

```
| Vocab          | Translation   |
| -------------- | ------------- |
| al fin y al cabo | in the end  |
```

### `/refresh-vocab` Command

Runs automatically via the Claude Code slash command. Steps:
1. Finds all class notes in `Class Notes/` not yet listed in `processed_notes.txt`
2. Generates a natural Spanish example sentence for each new word (via `claude haiku`)
3. Appends new rows to `Total Vocab.md`
4. Creates `spanish_new_cards.apkg` with only the new cards
5. Opens the `.apkg` in Anki for import

Skips duplicate words already present in `Total Vocab.md`.
